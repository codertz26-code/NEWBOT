Frequently Asked Questions

How do I upload my website? ▼
You have two options: 1) Upload a ZIP file containing all your website files (HTML, CSS, JS, images), or 2) Paste your HTML code directly in the text area. Both methods will give you a live link immediately.
Is it really free? ▼
Yes! SILA WEB HOSTING is completely free. You can host unlimited projects with no hidden costs. We provide free SSL certificates and 100MB storage per project.
What file types are supported? ▼
We support all standard web files: HTML, CSS, JavaScript, images (PNG, JPG, GIF, SVG), fonts, and more. For ZIP uploads, make sure your main file is named index.html.
How long do projects stay online? ▼
Projects stay online indefinitely as long as they are active. We don't delete projects unless they violate our terms of service or remain inactive for an extended period.
Can I use a custom domain? ▼
Yes! You can connect your own domain name to your hosted project. Contact us through WhatsApp or Telegram for assistance with custom domain setup.
Is there a file size limit? ▼
Yes, each project has a maximum file size of 100MB. If you need more space for larger projects, please contact us for special arrangements.
